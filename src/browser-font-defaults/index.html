Browser Font Defaults

Default values listed below are for the Chrome browser

h1 – Heading 1

h1 { font-size: 2em; margin-block: 0.67em; font-weight: bold; }
h1 { font-size: 32px; margin-block: 21.44px; font-weight: 700; }

h2 – Heading 2

h2 { font-size: 1.5em; margin-block: 0.83em; font-weight: bold; }
h2 { font-size: 24px; margin-block: 19.92px; font-weight: 700; }

h3 – Heading 3

h3 { font-size: 1.17em; margin-block: 1em; font-weight: bold; }
h3 { font-size: 18.72px; margin-block: 18.72px; font-weight: 700; }

h4 – Heading 4

h4 { margin-block: 1.33em; font-weight: bold; }
h4 { font-size: 16px; margin-block: 21.28px; font-weight: 700; }
h5 – Heading 5
h5 { font-size: 0.83em; margin-block: 1.67em; font-weight: bold; }
h5 { font-size: 13.28px; margin-block: 22.1776px; font-weight: 700; }
h6 – Heading 6
h6 { font-size: 0.67em; margin-block: 2.33em; font-weight: bold; }
h6 { font-size: 10.72px; margin-block: 24.9776px; font-weight: 700; }

Paragraph

p { margin-block: 1em; }
p { font-size: 16px; margin-block: 16px; font-weight: 400; }

Code
code { font-family: monospace; }
code { font-family: monospace; font-size: 13px; font-weight: 400; }

Link – example.com
a { color: -webkit-link; cursor: pointer; text-decoration: underline; }
a { font-size: 16px; font-weight: 400; }
